## 📊 **SQL Data Analytics Project**

### 📌 **Project Overview**

This project is a collection of SQL scripts designed to demonstrate core data analytics techniques used to extract insights, analyze trends, and evaluate business performance.

---

## 🔍 **Key Analytical Areas Covered**

### ⏳ **Time-Based Analysis**

* Analyzed data across different time periods (daily, monthly, yearly)
* Identified trends, seasonality, and growth patterns
* Used date functions and aggregations for time-series insights

---

### 📈 **Cumulative Metrics**

* Calculated running totals and moving averages
* Tracked performance over time using cumulative KPIs
* Applied **window functions** for advanced analytics

---

### 👥 **Customer Segmentation**

* Segmented data based on business rules (e.g., high-value customers, inactive users)
* Used conditional logic and grouping techniques
* Enabled targeted analysis for different customer groups

---

### 📉 **Performance Analysis**

* Evaluated key metrics such as revenue, transactions, and growth rates
* Compared performance across categories, regions, or time periods
* Identified top-performing and underperforming segments

---

## 🛠 **Tech Stack**

* SQL (T-SQL / MySQL)
* Window Functions
* Aggregations & Joins
* Case Statements & Subqueries

---

## 📂 **Project Structure**

* `time_analysis.sql` → Time-based trend analysis
* `cumulative_metrics.sql` → Running totals & KPIs
* `segmentation.sql` → Customer segmentation logic
* `performance_analysis.sql` → Business performance insights

---

## 💡 **Key Outcomes**

* Demonstrated strong SQL-based analytical capabilities
* Applied real-world data analysis techniques
* Generated actionable insights from raw datasets
* Strengthened understanding of business-driven analytics
